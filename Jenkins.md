# [Jenkins](https://jenkins.io/)

## 安装
CentOS7参考[下载页面](https://pkg.jenkins.io/redhat/)yum方式下载安装的Jenkins

安装路径
```bash
/etc/init.d/jenkins
/etc/logrotate.d/jenkins
# 配置文件
/etc/sysconfig/jenkins
# 工作目录
/usr/lib/jenkins
/usr/lib/jenkins/jenkins.war
/usr/sbin/rcjenkins
/var/cache/jenkins
/var/lib/jenkins
/var/log/jenkins
```

启动Jenkins  
sudo service jenkins start
